# 📅 2026-05-11 每日AI工具精选

## 1. re_gent：给AI Agent用的Git
- **简介**: 专为AI Agent设计的版本控制系统，追踪Agent的每一步操作、状态变化和决策过程，支持随时回滚。目前主要支持Claude Agent SDK。
- **链接**: [GitHub](https://github.com/regent-vcs/re_gent)
- **分类**: 开发工具

## 2. AlphaEvolve
- **简介**: DeepMind用Gemini驱动的编程Agent，能自己进化和优化算法，从数学问题到芯片设计、数据中心调度、AI训练优化都有突破。
- **链接**: [DeepMind Blog](https://deepmind.google/blog/alphaevolve-impact/)
- **分类**: Agent

## 3. Anthropic × SpaceX算力合作
- **简介**: Anthropic与SpaceX达成算力合作，Claude使用额度翻倍。SpaceX的Starlink卫星网络提供分布式计算资源。
- **链接**: [Anthropic Blog](https://www.anthropic.com/news/higher-limits-spacex)
- **分类**: 行业动态

## 4. Gemini API多模态文件搜索
- **简介**: Google给Gemini API新增多模态文件搜索功能，支持图片、PDF、表格等混合格式文档的统一检索，适用于企业级RAG应用。
- **链接**: [Google Blog](https://blog.google/innovation-and-ai/technology/developers-tools/expanded-gemini-api-file-search-multimodal-rag/)
- **分类**: 开发工具
